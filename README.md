# QBoss HTML 5 Theme

## Description
QBoss (kyo͞o-bŏs) is a clean and minimalistic HTML 5 theme built on [Kube](https://imperavi.com/kube/) CSS &amp; JS Framework.


## Supported Browsers
QBoss supports the latest, stable releases of all major browsers:

- Latest Chrome
- Latest Firefox
- Latest Safari
- Latest Opera
- Microsoft Edge
- Internet Explorer 11
